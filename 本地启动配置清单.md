# yu-ai-code-mother 本地启动配置清单

## 1. 本地启动前需要修改的配置

| 文件 | 配置项 | 需要你提供的信息 | 是否必填 | 说明 |
|---|---|---|---|---|
| `src/main/resources/application.yml` | `spring.datasource.url` | MySQL 地址、端口、数据库名 | 是 | 后端数据库连接 |
| `src/main/resources/application.yml` | `spring.datasource.username` | MySQL 用户名 | 是 | 数据库账号 |
| `src/main/resources/application.yml` | `spring.datasource.password` | MySQL 密码 | 是 | 数据库密码 |
| `src/main/resources/application.yml` | `spring.data.redis.host` | Redis 地址 | 是 | Redis 连接 |
| `src/main/resources/application.yml` | `spring.data.redis.port` | Redis 端口 | 是 | 一般是 `6379` |
| `src/main/resources/application.yml` | `spring.data.redis.password` | Redis 密码 | 视情况 | 没有密码可留空 |
| `src/main/resources/application.yml` | `langchain4j.open-ai.chat-model.api-key` | AI 平台 API Key | 是 | 聊天模型 Key |
| `src/main/resources/application.yml` | `langchain4j.open-ai.streaming-chat-model.api-key` | AI 平台 API Key | 建议 | 流式输出用 |
| `src/main/resources/application.yml` | `langchain4j.open-ai.reasoning-streaming-chat-model.api-key` | AI 平台 API Key | 视功能 | 推理模型用 |
| `src/main/resources/application.yml` | `langchain4j.open-ai.routing-chat-model.api-key` | AI 平台 API Key | 视功能 | 路由模型用 |
| `src/main/resources/application.yml` | 模型名 / `base-url` | 你使用的模型平台信息 | 是 | 例如 DeepSeek / DashScope |
| `src/main/resources/application.yml` | `server.port` | 本地服务端口 | 否 | 默认通常可直接用 |
| `src/main/resources/application.yml` | `server.servlet.context-path` | 接口上下文路径 | 否 | 默认一般是 `/api` |
| `src/main/resources/application.yml` | COS 存储配置 | `secretId`、`secretKey`、`region`、`bucket`、`host` | 视功能 | 文件上传 / 对象存储需要 |
| `src/main/resources/application.yml` | Pexels Key | Pexels API Key | 视功能 | 图片搜索相关 |
| `src/main/resources/application.yml` | DashScope 图像相关 Key | API Key | 视功能 | 图片生成等能力 |
| 前端环境文件（如 `.env*`）或前端请求配置 | `API_BASE_URL` | 后端地址 | 是 | 本地一般填 `http://localhost:8123/api` |
| 数据库初始化脚本 | 建库 SQL | 数据库名一致 | 是 | 要先导入表结构 / 初始数据 |

---

## 2. 最少需要准备的信息

如果你只是想先把项目跑起来，至少要准备以下 4 类信息：

| 类别 | 你要准备什么 |
|---|---|
| MySQL | 地址、端口、库名、用户名、密码 |
| Redis | 地址、端口、密码 |
| AI Key | 至少一套可用的模型平台 Key |
| 前端接口地址 | 指向本地后端，如 `http://localhost:8123/api` |
